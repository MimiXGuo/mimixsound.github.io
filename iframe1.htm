Flutter is an open-source UI toolkit created by Google for building natively compiled applications for Multiple Platforms from a single codebase(iOS, Android, Web, Desktop). It saves money and time.

High Performance that compiled to native machine (ARM) code, so they're typically more performant than those built with PhoneGap or React Native.

However, compared to other frameworks, Flutter apps can be larger in size, and community is smaller than React Native's, which can mean fewer resources and third-party libraries.
