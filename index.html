What are the main characteristics of a PWA(Progressive Web Application)?

1. Offline Capabilities: Progressive Web Application allows access even without internet, which means PWA can be functioning offline by using service worker to chache information and local storage. Besides, PWA can be open through a web browser and installed accessible via a URL or search engine as well as in app stores. It also build an app for all devices from one code.

2.Responsive Design: PWA can be adapt to different screen sizes and resolutions, with build an app for multiple devices from one code, including smartphones, tablets, laptops, and desktops.
